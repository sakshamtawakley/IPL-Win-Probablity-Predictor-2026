# IPL-Win-Probablity-Predictor-2026
This is a project I made in March-April 2026. I made some changes to it right now and fixed some bugs here and there.
Built a ML model to predict IPL match win probability using match-state features (runs/wickets/overs). Performed preprocessing feature engineering and scaling achieved ~80% accuracy (via Logistic Regression). Deployed via Streamlit for real-time interactive predictions,
Tech: Python; Scikit-learn; Pandas; Streamlit
